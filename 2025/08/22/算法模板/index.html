STL库里面的二分函数（upper_bound 、lower_bound）

  • lower_bound(first, last, value)
    • 返回指向区间 [first, last) 中第一个不小于 value 的位置（第一个 >= value）。
  • upper_bound(first, last, value)
    • 返回指向区间 [first, last) 中第一个严格大于 value 的位置（第一个 > value）。
lower_bound upper_bound
查找条件 第一个 不小于 value 的元素（第一个 >= value） 第一个 大于 value 的元素（第一个 > value）
返回值 指向第一个满足条件的元素（返回迭代器/指针） 指向第一个满足条件的元素（返回迭代器/指针）
边界 如果所有元素都小于 value，返回 last（区间末迭代器） 如果所有元素都小于等于 value，返回 last
示例 在 {1, 3, 6, 7, 8} 中查找 6，返回指向 6 的位置 在 {1, 3, 6, 7, 8} 中查找 6，返回指向 7 的位置
